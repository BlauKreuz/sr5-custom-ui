# sr5-custom-ui

For help and suggestions: [![Discord Server](https://img.shields.io/badge/-Discord-%232c2f33?style=flat-square&logo=discord)](https://discord.gg/NyCX2nN4qP)

Options to customize Shadowrun 5e user interface in Foundry VTT.

- Four optional color schemes in addition to keeping the default.
- Custom layouts, font changes etc. (includes new edit toggle and visual cliues for being in edit mode)
- Roll mode baked into 4 separate roll buttons in test dialogs (saves one click every time and reduces rol mistakes in wrong modes).
  -- hiding redundant roll mode buttons above message window.
- Rolled dice in new test cards are initially visible. New cards collapses older dice in cards.
- Hiding several tool-tips over attributes and other values.
- Delay for showing the roll cards after dice are rolled.
- Floating roll card duration.
